# 老黑起始页

一个基于Flutter的web仿终端起始页

## 项目启动
启动项目前需要安装chrome浏览器。
项目拉取到本地后，运行以下命令添加依赖
```sh
flutter pub get
```
成功后运行以下命令启动项目
```sh
flutter run -d chrome --web-renderer html
```

## 项目打包
```sh
flutter build web --web-renderer html
```
